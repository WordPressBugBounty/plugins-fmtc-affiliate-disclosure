Add FTC-Compliant Disclosure statement to the beginning of your blog posts

== Description ==

The FTC requires that bloggers, affiliates and publishers disclose their relationships with merchant if they earn money
from links within blog posts. This plugin will add a simple FTC-compliant disclosure statement at the beginning of all
blog posts.

== Installation ==

1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Can I change the message? =

Yes! Now you can (new in Version 2)

= Can I limit to which blog posts the disclosure is shown? =

Not now.

= Can I specify the disclosure to only appear on certain posts? =

Not now.
